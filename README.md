## About

This is a quick portfolio site, built with Hugo. The goal with this project was to get an initial version of the site up as quickly as possible - it took about 3.5 hrs including image editing and copywriting. The theme customisation is fairly minimal. 

## TIL

I sometimes have to fight perfectionism sometimes when I build things, particularly when they include personal information about me. My usual approach is to try and get an entire draft finished, since I find editing a lot easier, so I tried that approach here.
